# Inventory Management System (CLI)

A professional command-line interface (CLI) application built with **Python**, focusing on **Object-Oriented Programming (OOP)** principles and data persistence.

##  Features

- **Full CRUD Operations**: Create, Read, Update, and Delete inventory items.
- **Search Functionality**: Partial name matching to find products quickly.
- **Data Persistence**: Automatically saves and loads data using `JSON`.
- **Error Handling**: Robust input validation to prevent system crashes.
- **Low Stock Alerts**: Visual warnings for products with stock less than 5 units.

## Tech Stack

- **Language**: Python 3.x
- **Storage**: JSON (File-based database)
- **Concepts**: Object-Oriented Programming (Classes, Methods, Encapsulation)

##  Project Structure

- `main.py`: The core application logic and OOP structure.
- `inventory.json`: Persistent storage for product data.
- `README.md`: Project documentation.
